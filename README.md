## 🛠️ My Personal Touch

Besides following the tutorial, I:

- Added a difficulty selection before starting the game.
- Prevented the game from starting until a level is selected.
- Created different word lists for each difficulty level.
- Added a dynamic instructions section generated with JavaScript.
- Added **3 bonus seconds** for the first word.
- Refactored repeated code into reusable functions.
- Improved code organization and added descriptive comments.
- Saved the player's latest score using Local Storage.
